In the software industry, protecting intellectual property is extremely important. An unprotected binary is easy to analyze, which means innovations achieved through R&D can be easily stolen, paid applications can be compromised, and unauthorized modifications can be freely distributed.

Developed by the Stras4Ever group, Rusty Vault is a binary protection project designed to counter these threats and secure your applications.

Our Core Features

Our project focuses on several priority technical layers to complicate the work of attackers :

  • Packing: A basic packer designed for Rust source code or compiled binaries.
  • Static Obfuscation: This involves parsing the code, removing symbols, renaming variables, and encrypting strings to prevent simple reading.
  • Anti-Debug Security: We use available APIs and debugging indicators to detect analysis tools and virtual environments, such as VirtualBox, VMware, and x64dbg.

Advanced Protections

Depending on our development progress, Rusty Vault explores optional, high-level protection implementations:

  • Control Flow Flattening: A custom implementation to work with rustc and its different profiles, breaking the program's linear logic.
  • IAT/GOT Hooking: Parsing the IAT/GOT and modifying offsets at runtime.
  • Code Virtualization: Implementing a virtual machine to pack the binary.
